🍕 Pizza Shop Sales Performance Analysis (1-Year Overview)
📌 Project Overview

This project analyzes one year of pizza shop sales data to identify trends, customer preferences, and overall business performance. The goal is to transform raw transactional data into actionable insights that can support better business decisions.

🎯 Objectives
Analyze sales trends over time
Identify best-selling pizza types
Understand peak ordering periods
Evaluate revenue performance
Extract business insights from transactional data
🛠️ Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
📊 Key Insights
Sales vary significantly across different time periods
Certain pizza categories consistently outperform others
Peak ordering times reveal strong customer behavior patterns
A small number of products contribute a large portion of revenue
📁 Project Structure
Data cleaning and preprocessing
Exploratory data analysis (EDA)
Visualization of trends and patterns
